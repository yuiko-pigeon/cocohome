非同期 JavaScript ファイル (上級者向け)
async (非同期) フラグで読み込むべきローカルまたはサードパーティの JS ファイルをカンマ区切りで一覧にします。ここに追加すると、自分のサイトの JS ファイルは自動的に除外されます。 "Async Javascript" プラグインを使用すると、非同期 javascript の設定がより簡単で柔軟になります。 インストールして有効化するには、ここをクリック。
